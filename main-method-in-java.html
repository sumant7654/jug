Exploring Java Main Class Variations

1. Normal Main Class

public class Main {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}

Output: Hello World

2. Main Class with Default Access Modifier

default class Main {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}

Output: CE: modifier default not allowed here

3. Main Class with Protected Access Modifier

protected class Main {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}

Output: CE: modifier protected not allowed here

4. Main Class Without Access Modifier

class Main {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}

Output: Hello World

5. Main Class with Final Keyword

public final class Main {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}

Output: Hello World

6. Main Method Without Access Modifier

public class Main {
    static void main(String[] args) {
        System.out.println("Hello World");
    }
}

Output: CE: Main method not found in class Main, please define the main method as:
public static void main(String[] args)

7. Main Method with Final Keyword

public class Main {
    public final static void main(String[] args) {
        System.out.println("Hello World");
    }
}

Output: Hello World

8. Main Method with Strictfp Keyword

public class Main {
    public strictfp static void main(String[] args) {
        System.out.println("Hello World");
    }
}

Output: Hello World

9. Main Method with Different Order of Keywords

public class Main {
    strictfp final static public void main(String[] args) {
        System.out.println("Hello World");
    }
}

Output: Hello World

10. Main Method with Different Return Type

public class Main {
    strictfp final static public int main(String[] args) {
        System.out.println("Hello World");
        return 0;
    }
}

Output: RE: Main method must return a value of type void in class Main, please define the main method as:
public static void main(String[] args)

11. Main Method with Varargs

public class Main {
    strictfp static public void main(String... args) {
        System.out.println("Hello World");
    }
}

Output: Hello World

12. Main Method Overloaded by Varargs

public final class Main {
    public static void main(String args) {
        System.out.println("Hello World");
    }
    strictfp static public void main(String... args) {
        System.out.println("Hello Varargs");
    }
}

Output: Hello Varargs

13. Main Method Overloaded with Conflict

public final class Main {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
    strictfp static public void main(String... args) {
        System.out.println("Hello Varargs");
    }
}

Output: CE: cannot declare both main(String...) and main(String[]) in Main

14. Inheritance with Main Class

class Main {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}
public class Test extends Main {
}

Output: Hello World

15. Inheritance with Method Hiding

class Main {
    public static void main(String[] args) {
        System.out.println("Hello World-Main");
    }
}
public class Test extends Main {
    public static void main(String[] args) {
        System.out.println("Hello World-Test");
    }
}

Output: Hello World-Test

Conclusion

This document provides an overview of various Java main class structures and the corresponding outputs, showcasing the nuances of Java's main method declaration and its behavior under different scenarios. For more discussions and examples, feel free to join us at the Java User Group, Bhubaneswar!
